# Markdown 语法指南

Chaterm 知识库内置编辑器基于 Monaco（与 VSCode 同源），天然支持 Markdown 语法高亮与预览渲染。本文档列出了主要的 Markdown 语法及用法。

## 标题

使用 `#` 号表示标题，1-6 级依次递减：

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

## 文本样式

| 语法             | 效果         | 示例                |
| ---------------- | ------------ | ------------------- |
| `**粗体**`       | **粗体**     | `**重要内容**`      |
| `*斜体*`         | _斜体_       | `*强调内容*`        |
| `***粗斜体***`   | **_粗斜体_** | `***非常重要***`    |
| `~~删除线~~`     | ~~删除线~~   | `~~已废弃~~`        |
| `` `行内代码` `` | `行内代码`   | `` `npm install` `` |

## 列表

### 无序列表

使用 `-`、`*` 或 `+` 作为列表标记：

- 第一项
- 第二项
  - 嵌套项 A
  - 嵌套项 B
- 第三项

### 有序列表

使用数字加 `.` 作为列表标记：

1. 第一步
2. 第二步
3. 第三步

### 任务列表

- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 待办事项

## 代码

### 行内代码

使用反引号包裹：

执行 `ls -la` 查看文件列表

### 代码块

使用三个反引号包裹，可指定语言以获得语法高亮：

```python
def hello():
    print("Hello, World!")
```

支持的语言包括：`python`、`javascript`、`typescript`、`go`、`java`、`cpp`、`csharp`、`ruby`、`php`、`rust`、`sql`、`shell`、`json`、`yaml`、`markdown`、`html`、`css` 等。

## 引用

使用 `>` 表示引用：

> 这是一段引用文字。
> 可以跨越多行。
>
> 也可以包含空行。

嵌套引用：

> 外层引用
>
> > 内层引用

## 表格

使用 `|` 和 `-` 构建表格：

| 列1   | 列2   | 列3   |
| ----- | ----- | ----- |
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |

对齐方式：

| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 左     |    中    |     右 |

## 分隔线

使用三个或更多的 `-`、`*` 或 `_`：

---

---

---

## HTML

Markdown 中可以直接使用 HTML 标签（经安全过滤）：

<details>
<summary>点击展开</summary>

这里是折叠的内容。

</details>

<kbd>Ctrl</kbd> + <kbd>C</kbd>

## 转义字符

在特殊字符前加 `\` 可以取消其 Markdown 语义：

\*不是斜体\*
\# 不是标题
\[不是链接\](url)
